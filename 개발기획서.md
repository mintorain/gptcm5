## 1. 프로젝트 개요

### 1.1 목적
GPT 콘텐츠 메이커 **5기 모집을 위한 공개특강**을 진행한 상황에서,
**5기 수강 신청을 유도**하고, **공개특강 후기 작성에 대한 감사 선물을 제공**하기 위한 전용 랜딩페이지를 제작합니다.

> **핵심 포지셔닝**: 두 가지 목적을 동시에 달성하는 하이브리드 랜딩페이지
> - **1차 목표**: 5기 수강 신청 전환 (핵심 CTA)
> - **2차 목표**: 공개특강 후기 작성 → 감사 선물 제공 (보조 CTA)
>
> 공개특강 후기 선물을 통해 잠재 수강생과의 접점을 유지하고,
> 자연스럽게 5기 수강 신청으로 이어지도록 설계합니다.

### 1.2 대상 사용자
- **1차 대상**: 5기 모집 공개특강 참석자 (수강 신청 잠재 고객)
- **2차 대상**: 공개특강 다시보기를 통해 유입된 신규 방문자
- **3차 대상**: GPT 콘텐츠 메이커에 관심 있는 일반 방문자

### 1.3 공개특강 후기 작성 플랫폼

```
     ①②              ③④                  ⑦(1차CTA)      ⑤⑥(2차CTA)

심리학 원칙 매핑:
- 상호성(Reciprocity): 무료 공개특강 + 후기 선물 제공 → 수강 신청 유도
- 사회적 증거(Social Proof): 125명 누적 수료생, 바이브코딩 개발 사례
- 일관성(Consistency): 공개특강 참석 → 후기 작성 → 수강 신청 (단계적 몰입)
- 희소성(Scarcity): 2/19 자정 마감 카운트다운 → 긴급성
```

### 2.4 기존 페이지 강점·약점 분석 (CRO 관점)

#### 강점 (계승할 것)
- 사이버펑크 네온 테마의 강력한 시각적 차별화
- 다중 CTA 배치 (8곳) — 스크롤 어느 지점에서도 접근 가능
- 모바일 하단 고정 CTA — 전환 경로 최소화
- Google Forms 직접 임베드 — 페이지 이탈 방지
- 가격 투명성 + 오프라인 신청시 온라인 무료 인센티브

#### 약점 (개선할 것)
- 사회적 증거(Social Proof) 부족 → **후기 페이지에서 해결**
- 트래킹(GA, 전환 추적) 부재 → **기본 이벤트 추적 추가**
- 첫 화면 정보 과다 → **단일 메시지에 집중**
- CTA 8개 과다 → **이중 목표(수강 신청 + 후기 선물)로 CTA 집중, 플로팅 CTA는 1차 목표만**

| 카드 | 수치 | 아이콘 | 설명 |

**UX/UI 스펙**

| 요소 | 데스크탑 | 모바일 |

##### 구글시트 Apps Script 연동 — 기술 명세

> 기존 4기(`I:\gpt-content-maker\gpt_c_m_4`) 알고리즘을 기반으로 5기에 맞게 적용합니다.

**아키텍처 개요**

```
[랜딩페이지 (클라이언트)]
        │
        │  POST (FormData)
        │  - name: 이름
        │  - phone: 연락처
        │  - email: 이메일
        │  - source: "공개특강" 또는 "후기선물"
        │  - timestamp: 제출 시각 (자동)
        ↓
[Google Apps Script Web App]
        │
        │  doPost(e) 함수
        │  - 파라미터 파싱
        │  - 시트에 행 추가
        │  - 응답 반환
        ↓
[Google Sheets DB]
        │
        │  시트 구조:
        │  | 타임스탬프 | 이름 | 연락처 | 이메일 | 출처(source) |
        │
        ↓
[성공 응답 → 클라이언트 → 콘텐츠 언락]
```

**Google Apps Script 코드 (구글시트 측)**

```javascript
// Google Apps Script — Web App (doPost)
// 기존 4기 알고리즘 기반, 5기용 시트에 연결

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

    var name = e.parameter.name || '';
    var phone = e.parameter.phone || '';
    var email = e.parameter.email || '';
    var source = e.parameter.source || '';
    var timestamp = new Date();

    sheet.appendRow([timestamp, name, phone, email, source]);

    return ContentService
      .createTextOutput(JSON.stringify({ result: 'success' }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService
      .createTextOutput(JSON.stringify({ result: 'error', message: error.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// GET 요청 대응 (CORS 우회용)
function doGet(e) {
  return doPost(e);
}
```

```javascript
const YOUTUBE_URL = "https://youtu.be/BflL5wC4T-c";  // 5기 공개특강 유튜브 URL

  e.preventDefault();
  slStatus.textContent = "전송 중입니다...";

  const formData = new FormData(slForm);
  formData.append("source", "공개특강");

  try {
    const response = await fetch(SCRIPT_URL, {
      method: "POST",
      body: formData,
    });

    if (!response.ok) throw new Error("응답 오류");

    // 성공: 모달 닫기 + 영상 언락
    specialLectureModal.style.display = "none";

    slStatus.textContent = "전송 중 오류가 발생했습니다. 잠시 후 다시 시도해주세요.";
    slStatus.style.color = "var(--danger)";
  }
});

// 영상 언락 함수
function unlockVideo() {
  const container = document.getElementById("video-container");
  container.innerHTML = `
    <iframe
      src="https://www.youtube.com/embed/BflL5wC4T-c"
      frameborder="0"
      allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
      allowfullscreen
      style="position:absolute; inset:0; width:100%; height:100%;">
    </iframe>
  `;
  // 오버레이 텍스트 변경
  const overlay = document.getElementById("video-overlay");
  if (overlay) {
    overlay.innerHTML = "✔ 신청 완료 — 영상을 재생해주세요";
    overlay.style.color = "var(--neon-cyan)";
  }
  // 해당 섹션으로 스크롤

}
```
